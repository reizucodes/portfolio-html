# Setup Instructions

## Requirements
- Node.js
- npm

## Install
```bash
npm install
```

## Run Locally
```bash
npm run dev
```

## Production Build
```bash
npm run build
```

## Preview Build
```bash
npm run preview
```

## Relevant Files
- `src/App.vue`: page composition and section order.
- `src/data/content.ts`: portfolio content source.
- `src/types/content.ts`: shared content contracts.
- `src/components/sections/*`: portfolio sections.
